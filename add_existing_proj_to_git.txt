Adding existing project to Github

Create a new repository without any file (i.e. readme.txt, git.ignore)
Copy the new repository URL
Open command prompt
Navigate to the root directory of the project you want to push to git
initialize this project directory as a Git directory using: $ git init
Add all the files from this directory to the newly initialized local Git directory using: $ git add .
Commit the files: $ git commit -m “Initial commit”
Set the new repository created on Github on step 1 as the origin of the local git directory using: $ git remote add origin “repository URL from step 2 here”
Optional: to verify if the origin is correct: $ git remote -v
Pushes the changes from local repository to Github origin: $ git push origin master
